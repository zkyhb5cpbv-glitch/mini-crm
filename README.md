# Mini CRM

A lightweight, customizable CRM (Customer Relationship Management) system designed to help businesses manage customers, leads, contacts, and daily workflows in one place. This project focuses on providing a clean, efficient, and easy-to-use foundation that can be expanded with additional features as needed.

### Current Features

* Customer and contact management
* Lead tracking
* Notes and activity logging
* Search and filtering
* Simple dashboard
* Responsive user interface

### Goal

The goal of this project is to create a solid CRM foundation that developers and businesses can easily build upon. The architecture is designed to be modular, making it straightforward to add new functionality without rewriting the existing codebase.

### Feel Free to Build on It

This project is open for improvement and customization. Feel free to edit, modify, and build on top of it however you like. Whether you're adding authentication, analytics, email integration, task management, invoicing, AI features, or anything else, you're encouraged to make it your own.
